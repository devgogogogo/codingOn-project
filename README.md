## 🧠 개요
본 프로젝트는 배터리 사용 과정에서 축적되는 생애주기 데이터를 분석하여  
화재로 이어질 수 있는 위험 신호가 언제부터 나타나는지를 사전에 파악하는 것을 목표로 했습니다.

배터리의 충전·방전 조건, 사용 누적 상태, 내부 특성 변화가 시간에 따라 어떻게 달라지는지를 시계열 데이터로 분석하고,  
위험 점수 변화와의 관계를 통해 화재 위험이 급격히 증가하는 기준 지점을 도출하고자 했습니다.

이를 통해 배터리 안전 관리 기준 설정에 활용 가능한 데이터 기반 근거를 제시하는 것을 목표로 합니다.

---

## 🎯 프로젝트 목표
- 배터리 생애주기 데이터 기반 이상 징후 탐지
- 화재 위험 점수가 급격히 상승하는 임계 시점 도출
- 위험 증가에 영향을 주는 핵심 사용 조건 식별
- 배터리 안전 관리 기준 설정을 위한 정량적 근거 확보

---

## 📊 데이터 구성
- **총 데이터 수**: 약 47만 건
- **배터리 종류**
  - Li-ion NMC
  - LFP
  - 전고체 배터리
- **샘플 구성**
  - 배터리 종류별 생애주기 시계열 샘플 각 400개 (총 1,200개)

### 주요 변수
- **독립 변수**
  - 사이클 수
  - 충전율 / 방전율
  - 셀 온도
  - 내부 저항
  - 용량 유지율
  - 급속 충전 노출 횟수
  - 누적 고온 사이클
  - 가역적 손상 지수
- **종속 변수**
  - 화재 위험 점수(Risk Score)

## 📌 결과 및 인사이트
- 급속 충전 노출 횟수가 화재 위험 점수 상승과 가장 강한 연관성을 보임
- 충전율 1.5C 이상 조건에서 위험 점수 급격히 증가
- 약 400 사이클 이후 위험 점수 변동성 급증
- 내부 저항 증가와 비가역적 손상이 위험 상승의 핵심 요인으로 확인됨

## 💬 향후 개선 방향
- 실시간 배터리 데이터 연동을 통한 위험 예측 모델 확장
- 이상 탐지 알고리즘 고도화
- 배터리 관리 시스템(BMS) 적용 가능성 검토
- 다양한 환경 조건(온도, 사용 패턴) 추가 분석
